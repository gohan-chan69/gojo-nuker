# gojo-nuker
a fast nuker with nice features
## Installation And Use
- Click on the `gojo-nuker.zip`.
- Extract the zip.
- Open `cmd`.
- Open main.py with `py main.py` command 
- Enter your user token or bot token in config.json or in the code.
